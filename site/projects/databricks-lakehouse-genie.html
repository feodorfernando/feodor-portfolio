What it is

A complete Databricks lakehouse: Auto Loader incrementally ingests raw files to a Delta bronze table; a MERGE-based silver transform cleans and deduplicates; gold builds BI-ready aggregates. The differentiator is the Genie semantic layer — business-language column comments and a certified metric view — which is the single biggest lever on natural-language-to-SQL accuracy.

Design & architecture

How I built it

  • Auto Loader incremental ingest with schema evolution + checkpoints; trigger(availableNow=True) runs batch-style on new files only.
  • Silver MERGE: type casting, quality filters, latest-per-key upsert (idempotent), then OPTIMIZE … ZORDER for read performance.
  • Gold + semantic layer: annotate every column in business language and publish a certified metric view so Genie answers from curated logic instead of guessing — the highest-ROI step for NL-to-SQL accuracy.
